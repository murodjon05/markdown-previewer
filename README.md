## ‚ú® Features

- **Live Preview**: See your Markdown rendered in real-time as you type
- **Full Markdown Support**: Headings, bold, italic, links, lists, code blocks, blockquotes, and more
- **Dark/Light Mode**: Toggle between themes with saved preference
- **Scroll Synchronization**: Editor and preview scroll together
- **Copy HTML**: Export rendered HTML to clipboard
- **Responsive Design**: Works on desktop and mobile devices
- **XSS Protection**: HTML is sanitized to prevent security vulnerabilities
